# Menu Builder (Roadmap)

This folder tracks research, design specs, and implementation notes for DisplayDeck's dynamic Menu Builder feature.

## Phases

### Phase 1: Core Menu System ✅
Templates, theme tokens, section/item CRUD, player rendering.
See `PHASE_1.md` for implementation details.

### Phase 2: QSR Templates & Enhanced Features ✅
Professional fast-food style templates (QSR Board, Drive-Thru), badge system, variant pricing, combo support, enhanced theme controls.
See `PHASE_2.md` for implementation details.

### Phase 3: Integration & Automation (Planned)
- POS system integration
- Scheduled price changes
- CSV/API bulk import

### Phase 4: Advanced Builder (Planned)
- Freeform canvas designer
- Drag-and-drop positioning
- Custom animations

## Available Templates

| Template | Description | Best For |
|----------|-------------|----------|
| `classic` | Clean sections with cards | Restaurants, cafes |
| `minimal` | Simple, elegant design | Modern venues |
| `neon` | Glowing effects | Entertainment, nightlife |
| `qsr` | Large images, price badges | Fast food, takeaway |
| `drivethru` | Max legibility, numbered | Outdoor displays |

## Quick Start

1. Create a menu in Dashboard → Menus
2. Select a template (qsr/drivethru for fast food)
3. Choose a color palette from "Suggested palettes"
4. Add sections (e.g., "Burgers", "Drinks")
5. Add items with images and prices
6. Preview and assign to displays
