## Technical Specifications

### Image Requirements

**iOS App Store**:
- Format: PNG or JPEG
- Color Space: sRGB or P3
- Resolution: 72 DPI minimum
- No alpha transparency for screenshots
- No rounded corners (will be added automatically)

**Google Play Store**:
- Format: PNG or JPEG  
- Resolution: 320 DPI maximum
- File size: 8MB maximum per image
- No borders, logos, or promotional text overlay

### Screenshot Dimensions by Device

**iPhone Screenshots**:
- iPhone 14 Pro Max (6.7"): 1290 x 2796 pixels
- iPhone 11 Pro Max (6.5"): 1242 x 2688 pixels
- iPhone 8 Plus (5.5"): 1242 x 2208 pixels

**iPad Screenshots**:
- iPad Pro 12.9" (3rd gen+): 2048 x 2732 pixels (portrait)
- iPad Pro 12.9" (3rd gen+): 2732 x 2048 pixels (landscape)

**Android Screenshots**:
- Minimum: 320 x 480 pixels
- Maximum: 3840 x 3840 pixels  
- Recommended: 1080 x 1920 pixels (portrait)
- Recommended: 1920 x 1080 pixels (landscape)
